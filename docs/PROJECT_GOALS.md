Primary Goal

Build a lightweight TypeScript library for creating physics educational
graphics that can be embedded directly into the Django Question Bank.

Success Criteria

✓ Draw physics figures

✓ Draw graphs

✓ Draw tables

✓ Export to SVG

✓ Export to PNG

✓ Easy integration with Django

✓ Easy API



Non Goals

✗ Physics simulation

✗ Game engine

✗ CAD software

✗ 3D rendering
